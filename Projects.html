Projects & Problems

The following examples highlight technical projects, troubleshooting scenarios, and infrastructure work I have supported while working onsite as an IT Support Technician.

Critical Nurse Call System Restoration

Responded to a high-priority outage involving a nurse call system responsible for alerting healthcare staff when residents required assistance.

Deployed pre-imaged laptops, installed and configured vendor call bell software, coordinated with MSP engineers to provision user accounts, validate Group Policy application, and configure systems for continuous 24/7 operation.

Restored mission-critical resident-to-staff communication while permanent remediation was developed.

Active Directory to Entra ID / Intune Migration Support

Supporting migration from on-premises Active Directory Domain Services to Microsoft Entra ID and Intune cloud-based management.

Developing technical documentation covering imaging workflow changes, enrollment procedures, deployment standards, and troubleshooting processes during phased rollout.

Helping improve migration consistency, reduce deployment issues, and streamline technician onboarding to the new environment.

Fiber Router Cutover & Connectivity Migration

Assisted with cutover from legacy routing infrastructure to new fiber-backed network hardware supporting resident cable and television services.

Worked alongside network engineers to validate connectivity, verify routing behavior, and troubleshoot service-impacting issues during migration.

Helped ensure successful infrastructure transition with minimal resident downtime.

Apple Device Enrollment & MDM Deployment

Configured and enrolled iPhones and iPads into Microsoft Intune and Apple Business Manager for enterprise mobile device management.

Used Apple Configurator to manually provision and prepare devices for organizational deployment.

Troubleshot Apple Business Manager and enrollment issues alongside MSP engineers to resolve provisioning, assignment, and deployment problems.

Cisco IP Phone Firmware Troubleshooting

Diagnosed intermittent Cisco IP phone failures affecting communication with residents’ families and healthcare providers.

Identified firmware incompatibility between deployed phone firmware and legacy phone system infrastructure.

Collaborated with vendor engineer to stage correct firmware on server infrastructure and coordinate deployment/update process.

Restored reliable phone functionality and eliminated recurring communication disruptions.

UPS Replacement & Network Monitoring Upgrade

Replaced a failing legacy UPS supporting an older phone system with a new UPS unit.

Installed a network interface card so the new UPS could be monitored on the network.

After migrating the connected equipment, verified all services were restored and confirmed normal operation with staff.
