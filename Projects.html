Projects & Problems

The following examples highlight projects I have supported and problems I have helped solve while working onsite as an IT Support Technician at LutherCare Retirement Community.

Critical Call System Restoration (Healthcare Environment)

In one instance, I was faced with a high-priority issue where the nurse call system—responsible for alerting staff when residents required assistance—suddenly stopped functioning. Recognizing the immediate safety risk, I had to act quickly to implement a temporary but reliable solution.

I deployed two pre-imaged laptops and configured them with the necessary call bell software to restore functionality at the nursing stations. I then collaborated with engineers at the MSP to create and assign appropriate user accounts, ensure proper group policy application, and configure the systems for continuous 24/7 operation.

This solution allowed staff to maintain critical communication with residents while a long-term fix was developed.

Active Directory to Entra ID & Intune Transition Support

I am currently assisting with a transition from an on-premises Active Directory Domain Services (AD DS) environment to a cloud-based solution using Microsoft Entra ID and Intune.

As part of this effort, I am developing detailed how-to documentation to support engineers and technicians through the transition. This includes outlining changes to the imaging and deployment process for new machines, as well as documenting troubleshooting steps for issues encountered during the soft rollout.

My role helps ensure a smoother migration by improving consistency, reducing errors, and supporting both technical staff and end users throughout the process.

Network Cutover & Connectivity Migration (Fiber Upgrade)

I assisted in the cutover process of migrating network connectivity from an existing router to a new fiber-backed solution that supports cable services for residents. This system is critical, as it provides television access across the facility.

During the transition, I worked alongside network engineers to ensure a smooth switchover with minimal disruption to residents. This included helping validate connectivity, confirming proper routing, and troubleshooting any issues that arose during the migration process.

My involvement helped ensure a seamless transition to the new infrastructure while maintaining reliable service for end users.

Cisco Phone System Troubleshooting & Firmware Resolution

I investigated a recurring issue where Cisco IP phones used for communication with residents’ families and healthcare providers would intermittently stop functioning unless manually configured with an extension.

After analyzing the behavior, I identified that the root cause was a firmware incompatibility—the phones were running a newer firmware version that was not fully supported by the facility’s legacy phone system. I collaborated with a vendor engineer to implement a solution by staging the appropriate firmware on a server and coordinating the update process using network management protocols.

This resolution restored consistent phone functionality and improved reliability for critical communications within the facility.
